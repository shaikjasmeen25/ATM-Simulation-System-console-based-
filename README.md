# ATM Simulation System (Java)

## Description
A console-based ATM application built using Core Java.  
The system allows users to authenticate using a PIN and perform basic banking operations.

## Features
- PIN authentication
- Check balance
- Deposit money
- Withdraw money
- Exit option

## Technologies Used
- Java
- OOP concepts
- Scanner class

## How to Run
1. Open the project in VS Code
2. Compile ATM.java
3. Run the program
4. Enter PIN: 1234
